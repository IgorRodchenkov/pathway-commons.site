Citations were downloaded from Scopus (http://www.scopus.com/)

The article used as the seed was 

Ethan Cerami's 2011 article entitled "Pathway Commons, a web resource for biological pathway data" 

http://www.scopus.com/record/display.url?eid=2-s2.0-78651332286&origin=resultslist&sort=plf-f&src=s&st1=pathway+commons+cerami&sid=9E090C1A7D9EA69387BB574A3ACFF43A.kqQeWtawXauCyC8ghhRGJg%3a100&sot=q&sdt=b&sl=42&s=TITLE-ABS-KEY-AUTH%28pathway+commons+cerami%29&relpos=4&relpos=4&citeCnt=106&searchTerm=TITLE-ABS-KEY-AUTH%28pathway+commons+cerami%29

Citations were downloaded using the "View details of all NUMBER citations" link. On the following page the "All" box was selected, then clicking the "Export" link. On the "Output: Export, Print, E-mail or Create a Bibliography" page, the export format used was "Comma separated file, .csv (e.g. Excel)" outputting "Citations Only"
